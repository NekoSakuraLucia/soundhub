# SoundHub 🎵

> A powerful Discord music bot built with discord.js and discord-player.

---

## ✨ Features

- 🎵 **High-quality YouTube playback**
- 📋 **Queue management system**
- ⏯️ **Rich playback controls**
- 🎨 **Beautiful embeds**
- ⚡ **Fast and responsive**
- 🔐 **Permission handling**

---

## 🎮 Commands

| Command    | Description        |
|------------|--------------------|
| `/play`    | Play a song        |
| `/pause`   | Pause track        |
| `/resume`  | Resume playback    |
| `/stop`    | Stop playing       |
| `/skip`    | Skip track         |
| `/queue`   | View queue         |
| `/exit`    | Disconnect bot     |

---

## 🚀 Setup

### Prerequisites

- Node.js 18+
- Discord Bot Token
- Discord Server

### Installation

```bash
git clone https://github.com/yourusername/soundhub.git
cd soundhub
npm install
```

### Configuration
Create a .env file:
```bash
DISCORD_TOKEN=your_bot_token
CLIENT_ID=your_client_id
GUILD_ID=your_server_id
```

### Start
```bash
node index.js
```
